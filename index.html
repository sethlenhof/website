    Related Skills

    Description

    Dequeue is our team 's submission for a semester-long hackathon organized by KnightHacks. The idea was inspired by the Stream Deck, a device used by streamers to visually manage computer shortcuts. Instead of memorizing numerous shortcuts or purchasing an expensive device like the Stream Deck, Dequeue enables you to display and quickly access shortcuts directly on your phone. The mobile app for Dequeue was developed using React Native, while the desktop app was built with Electron, ensuring compatibility across multiple platforms. All data is stored locally on the computer and communicated to the phone via LAN.

    • Created an electron app for desktop to allow users to create commands to run on computer

    • Designed and implemented mobile app using React Native that communicates with the desktop app

    • Utilized Node.js based web sockets to relay commands to the desktop based on users input on the mobile app
